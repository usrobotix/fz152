# fz152

Репозиторий для разработки пакетов документов по персональным данным согласно 152-ФЗ «О персональных данных».

## Структура репозитория

| Папка | Описание |
|-------|---------|
| `Законы/` | Актуальные редакции нормативных правовых актов |
| `cdp26/` | Пакет документов для клиента cdp26 |
| `ао курорт нальчик/` | Пакет документов для клиента АО Курорт Нальчик |
| `диджитал плюс/` | Внутренние документы нашей организации |
| `Глобальный/` | Универсальный полный пакет шаблонов (независимо от отрасли и наличия сайта) |
| `чек-лист/` | Брифование клиентов и чек-листы проверки наличия документов |
| `черновики/` | Временная рабочая область (см. [`черновики/README.md`](черновики/README.md)) |

## Отслеживание статусов документов

В каждой папке находится файл `STATUS.md` с таблицей документов, их версиями и текущим статусом.  
Центральный индекс всех статусных файлов: **[STATUS_INDEX.md](STATUS_INDEX.md)**

### Обязательный рабочий процесс

> **Перед началом любой задачи** — откройте соответствующий `STATUS.md` и ознакомьтесь с текущим состоянием документов.  
> **В ходе работы** — обновляйте `STATUS.md` при добавлении или изменении любого документа.  
> **Перед открытием PR** — убедитесь, что все затронутые `STATUS.md` обновлены.
