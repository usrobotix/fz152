# fz152

## Статусы и черновики

- В корне репозитория ведется `STATUS.md` — **индекс + агрегатор**.
- В каждой ключевой папке ведется свой `STATUS.md` со статусами документов.
- В каждой ключевой папке есть подпапка `Черновики/`:
  - всё, что в `Черновики/`, не считается финальным;
  - после завершения документ переносится в целевую папку (вне `Черновики/`), и это фиксируется в статусах.

Правило работы:
- Перед началом любой задачи и перед PR/агентами обновляйте статусы в затронутых папках.

См. также: `STATUS.md`.
