# Сайт — Документы для размещения на сайте

Здесь размещаются **финальные** (утверждённые) документы, предназначенные для публикации на сайте https://liumi.ru.

Типичный состав:
- Политика в отношении обработки персональных данных (публичная версия)
- Политика использования Cookies
- Условия пользования сайтом
- Согласие на публикацию отзывов
- Приказ об утверждении Политики ПДн

> Черновики и рабочие версии хранятся в `лиу ми/Черновики/Документы для размещения на сайте/`. Переносите документ сюда только после согласования и утверждения.
