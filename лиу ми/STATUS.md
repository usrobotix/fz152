# Workflow

## Финальные документы
- Все финальные документы находятся в `лиу ми/<Группа>/`.
- Пакет для публикации на сайте: `лиу ми/Сайт/`

## Черновики
- Все черновики находятся в `лиу ми/Черновики/`.
- Если используются черновики “для сайта”: `лиу ми/Черновики/Документы для размещения на сайте/`
