# Статус файлов — Черновики

Папка содержит временные рабочие файлы, создаваемые в ходе разработки. После финализации файлы копируются в соответствующие целевые папки.

## Таблица файлов в работе

| Файл | Описание / назначение | Дата создания | Статус | Целевая папка | Примечания |
|------|-----------------------|--------------|--------|--------------|-----------|
| _(добавьте файл)_ | | | в работе / перенести в … | | |

## Статусы

- **в работе** — файл ещё в разработке
- **перенести в …** — файл готов, ожидает копирования в целевую папку
- **перенесён** — финальная версия скопирована в целевую папку (строку можно удалить)

## Правило обновления

> **Обновляйте этот файл каждый раз, когда в папке добавляется, изменяется или переносится любой файл, а также перед открытием PR.**
