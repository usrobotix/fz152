# Черновики

Эта папка является **временной рабочей областью** для файлов, создаваемых в ходе разработки.

## Правила работы

1. Все временные, промежуточные и рабочие файлы создаются здесь.
2. После доведения файла до финальной версии **скопируйте его в соответствующую целевую папку** (например, `cdp26/`, `Глобальный/`, `диджитал плюс/` и т.д.).
3. В `черновики/` хранятся только рабочие версии; финальные документы здесь не хранятся.
4. Перед открытием PR обновите [`STATUS.md`](STATUS.md) в этой папке.

## Целевые папки

| Тип документа | Целевая папка |
|--------------|--------------|
| Пакет для клиента cdp26 | `cdp26/` |
| Пакет для клиента АО Курорт Нальчик | `ао курорт нальчик/` |
| Внутренние документы | `диджитал плюс/` |
| Универсальные шаблоны | `Глобальный/` |
| Брифы и чек-листы | `чек-лист/` |
| Нормативные акты | `Законы/` |
