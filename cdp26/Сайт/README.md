# Сайт — Документы для размещения на сайте

Здесь размещаются **финальные** (утверждённые) документы, предназначенные для публикации на сайте организации.

Типичный состав:
- Политика в отношении обработки персональных данных (публичная версия)
- Политика использования Cookies
- Условия пользования сайтом
- Согласие на публикацию отзывов
- Cookie-баннер (текст/требования к реализации)
- Согласие на обработку ПДн через формы сайта

> Черновики и рабочие версии хранятся в `cdp26/Черновики/Документы для размещения на сайте/`. Переносите документ сюда только после согласования и утверждения.
