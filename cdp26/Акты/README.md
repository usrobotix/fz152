# Акты

Здесь размещаются **финальные** (утверждённые/подписанные) акты.

Типичный состав:
- Акт определения уровня защищённости ПДн в ИСПДн
- Акт оценки вреда субъектам ПДн
- Акт уничтожения СКЗИ
- Акт об уничтожении съёмных машинных носителей ПДн
- Акт об уничтожении персональных данных сотрудника

> Черновики и рабочие версии хранятся в `cdp26/Черновики/`. Переносите акт сюда только после подписания всеми ответственными лицами.
