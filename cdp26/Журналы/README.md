# Журналы — Эксплуатационные журналы

Здесь размещаются **финальные** (утверждённые/введённые в эксплуатацию) формы и заполненные журналы.

Типичный состав:
- Журнал учёта съёмных машинных носителей ПДн
- Журнал поэкземплярного учёта СКЗИ
- Журнал учёта ключей / карт доступа / ключей хранилищ / личных печатей
- Журнал учёта нештатных ситуаций / вскрытия / опечатывания / профработ / модификаций
- Журнал учёта обращений субъектов ПДн
- Журнал учёта записей резервных копий

> Черновики форм и рабочие версии хранятся в `cdp26/Черновики/`. Переносите форму сюда только после утверждения и принятия в эксплуатацию.
