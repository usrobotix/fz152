# Черновики

Эта папка предназначена для временных/рабочих материалов.

Правило:
- Документ **не считается готовым/финальным**, пока он находится в `Черновики/`.
- По завершении работы документ **переносится** в целевую папку (вне `Черновики/`), а запись о переносе фиксируется в `STATUS.md`.

Дополнительно:
- В `Черновики/STATUS.md` фиксируем, что именно в работе и **куда переносить**.

## Принцип “единый источник” для Политики ПДн и Приказа

**Важно:** не дублируем полный текст Политики в приказе.

Причина: Политика ведётся отдельным документом (`cdp26/Черновики/Документы для размещения на сайте/Политика.md`, целевой файл для сайта — `cdp26/Сайт/Политика_обработки_ПДн.md`).  
Если вставлять полный текст Политики в “Приложение №1” приказа, версии начинают расходиться.

**Как делаем правильно:**
- В приказе: “Утвердить Политику … (Приложение №1)”.
- В Приложении №1: заголовок Политики + указание на файл, где находится актуальный текст (без копирования всего документа).
