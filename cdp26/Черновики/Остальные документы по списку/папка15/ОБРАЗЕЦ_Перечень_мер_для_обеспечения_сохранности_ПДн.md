Перечень мер, необходимых для обеспечения условий, обеспечивающих сохранность персональных данных и исключающих несанкционированный к ним доступ, а также лиц, ответственных за реализацию указанных мер.

|     |     |     |
| --- | --- | --- |
| **Категория мер** | **Меры** | **Ответственные лица** |
| Организационные | Назначение ответственного за обработку персональных данных (издаётся приказ руководителя). | Главный врач (или лицо, назначенное приказом). |
|     | Регистрация событий и логирование действий пользователей, ведение журналов доступа к данным. | IT-отдел, ответственное лицо. |
|     | Ознакомление сотрудников с обязанностями в области работы с персональными данными под подпись, внесение изменений в должностные инструкции или трудовые договоры при необходимости. | Ответственное лицо, юрисконсульт, отдел кадров. |
| Технические | Шифрование персональных данных при передаче через интернет и хранении в электронных системах. | IT-отдел, ответственное лицо. |
|     | Внедрение средств защиты от несанкционированного доступа (системы идентификации и аутентификации, разграничение прав доступа). | IT-отдел, ответственное лицо. |
