# ПДн — Персональные данные

Здесь размещаются **финальные** (утверждённые) документы по обработке и защите персональных данных.

Типичный состав:
- Приказы об утверждении Политики, Положения, Перечней по ПДн
- Приказы о назначении ответственных за обработку ПДн
- Приказ о создании комиссии по защите ПДн
- Приказ об организации режима безопасности помещений
- Приказ об утверждении правил оценки вреда субъектам ПДн
- Приказ об утверждении регламента внутреннего контроля
- Формы: согласия, отзывы, уведомления

> Черновики и рабочие версии хранятся в `cdp26/Черновики/`. Переносите документ сюда только после согласования и утверждения.
