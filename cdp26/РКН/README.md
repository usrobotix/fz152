# РКН — Роскомнадзор

Здесь размещаются **финальные** (утверждённые) документы для взаимодействия с Роскомнадзором.

Типичный состав:
- Проект уведомления РКН о намерении осуществлять обработку ПДн
- Проект уведомления РКН об изменении сведений
- Инструкция по подаче уведомления в Роскомнадзор
- Рекомендации по заполнению формы уведомления

> Черновики и рабочие версии хранятся в `cdp26/Черновики/`. Переносите документ сюда только после согласования и утверждения.
