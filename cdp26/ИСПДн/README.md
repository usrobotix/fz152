# ИСПДн — Информационные системы персональных данных

Здесь размещаются **финальные** (утверждённые) документы по защите ПДн в информационных системах.

Типичный состав:
- Приказ о назначении ответственного за обеспечение безопасности ПДн в ИСПДн + инструкция
- Приказ об утверждении инструкций по защите ПДн (пользователь ИСПДн, парольная, антивирусная, съёмные носители)
- Приказ об утверждении регламента реагирования на инциденты ИБ
- Приказ о разрешительной системе доступа + положение
- Приказ о проведении сканирования на уязвимости
- Акт определения уровня защищённости ПДн (для каждой ИСПДн)

> Черновики и рабочие версии хранятся в `cdp26/Черновики/`. Переносите документ сюда только после согласования и утверждения.
