# СКЗИ — Средства криптографической защиты информации

Здесь размещаются **финальные** (утверждённые) документы по обращению со СКЗИ.

Типичный состав:
- Приказ об обращении со СКЗИ + приложения (инструкция, перечень режимных помещений, хранилища, допущенные)
- Акт уничтожения СКЗИ
- Журнал поэкземплярного учёта СКЗИ

> Черновики и рабочие версии хранятся в `cdp26/Черновики/`. Переносите документ сюда только после согласования и утверждения.
