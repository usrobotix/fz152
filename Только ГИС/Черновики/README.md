# Черновики

Эта папка предназначена для временных/рабочих материалов.

Правило:
- Документ **не считается готовым/финальным**, пока он находится в `Черновики/`.
- По завершении работы документ **переносится** в целевую папку (вне `Черновики/`), а запись о переносе фиксируется в `STATUS.md`.

Дополнительно:
- В `Черновики/STATUS.md` фиксируем, что именно в работе и **куда переносить**.
