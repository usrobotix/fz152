# STATUS (индекс + агрегатор)

## Правила
- Перед началом любой работы по документам **сверяйтесь** со статусами в соответствующей папке и **обновляйте** их.
- Перед созданием PR / запуском агентов / автоматизаций — **обновите** `STATUS.md` в затронутых папках.
- Все рабочие материалы держим в `Черновики/`. Финальным документ считается только после переноса из `Черновики/`.
