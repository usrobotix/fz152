# Статус документов — чек-лист

Папка содержит материалы для брифования клиентов по отраслям и чек-листы проверки наличия документов в зависимости от требований законодательства и статуса компании.

## Таблица документов

| Документ | Назначение | Версия / Дата | Статус | Ответственный | Примечания |
|----------|-----------|--------------|--------|--------------|-----------|
| _(добавьте документ)_ | | | черновик / на согласовании / финал | | |

## Статусы

- **черновик** — документ в разработке
- **на согласовании** — на внутреннем согласовании
- **финал** — утверждён, готов к использованию

## Правило обновления

> **Обновляйте этот файл каждый раз, когда в папке добавляется или изменяется любой документ, а также перед открытием PR.**
